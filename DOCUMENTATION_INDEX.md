## 🔑 Conceitos Principais

### Padrão de Uso
```
1. Ler documentação
2. Configurar Supabase
3. Executar aplicação
4. Acessar /admin
5. Usar funcionalidades
6. Estender conforme necessário
```

### Estrutura Típica de Página
```
Page (/admin/places)
  ↓
AdminLayout (layout com sidebar)
  ↓
PageHeader (título e ação)
  ↓
Componentes (Cards, Tabelas, Formulários)
  ↓
Serviços (admin.service.ts)
  ↓
Supabase (Database + Auth + Storage)
```

### Fluxo de Dados
```
Usuário → UI Component → Serviço → Supabase → Banco
Banco → Serviço → UI Component → Usuário
```
